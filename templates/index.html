Algerian Forest Fires Dataset

A comprehensive analysis of forest fire patterns in Algeria

About the Dataset

The dataset includes 244 instances from two regions in Algeria: Bejaia Region in the northeast and Sidi Bel-Abbes Region in the northwest. Collected between June 2012 and September 2012, it classifies data into fire (138 instances) and not fire (106 instances) classes. It has 11 attributes and one output attribute (class).

Attribute Information:

  • Date: Day, month ('June' to 'September'), year (2012)
  • Temp: Temperature max (22 to 42 °C)
  • RH: Relative Humidity (21% to 90%)
  • Ws: Wind Speed (6 to 29 km/h)
  • Rain: Total rain (0 to 16.8 mm)
  • FWI Components: FFMC, DMC, DC, ISI, BUI, FWI indices
  • Classes: Two classes: Fire and Not Fire

Step-by-Step Process

1. Data Cleaning

Removed null values, fixed column name spacing, and converted necessary features to integer and float types.

2. Exploratory Data Analysis (EDA)

Dropped date-related columns, encoded classes, and prepared the data for analysis.

3. Visualization

Created density plots for all features, a pie chart for class distribution, and a heatmap to study correlations. Conducted monthly fire analysis using count plots.

4. Train-Test Split

Split the features into dependent and independent variables and applied a train-test split using the scikit-learn library.

5. Feature Selection

Selected features based on correlation. Removed multicollinear features to improve model accuracy.

6. Feature Scaling

Applied standardization using a Standard Scaler. Created box plots to understand the effect of scaling.

7. Regression Models

Evaluated Linear, Lasso, Ridge, and ElasticNet regression models with cross-validation.

8. Ridge Regression

Chose Ridge Regression as the best model based on its superior R² score.

9. Deployment

Created a pickle file for the model, developed a Flask app, and deployed it on AWS Elastic Beanstalk.
